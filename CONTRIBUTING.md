# Contributing to Windows Vision

Thank you for your interest in contributing to Windows Vision!

## How to Contribute

### 1. Report Bugs
- Use GitHub Issues
- Include: Windows version, error messages
- Steps to reproduce

### 2. Suggest Features
- Check Roadmap first
- Explain use case
- Suggest implementation

### 3. Submit Code
- Fork repository
- Create feature branch
- Write clear commits
- Add tests
- Submit Pull Request

## Code Structure

```
src/
├── vision-simple.bat
├── vision-fixed.ps1
└── windows-vision-clean.ps1

docs/
├── INSTALLATION.md
└── API_REFERENCE.md

tests/
└── run-tests.ps1
```

## Testing

Before submitting:
```powershell
.\tests\run-tests.ps1
```

## Code Style
- Indentation: 4 spaces
- Naming: PascalCase functions
- Comments: English preferred
- Error handling: Try/Catch
